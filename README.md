# 📈 Análise de Ações da Bolsa Brasileira com Python

Este projeto utiliza Python para coletar e visualizar dados históricos de ações da bolsa brasileira (B3).

## Bibliotecas utilizadas

* pandas
* yfinance
* matplotlib
* scikit-learn

## O que o código faz

* Baixa dados históricos de ações da B3
* Mostra o preço de fechamento das ações
* Normaliza os dados para análise

## Ações utilizadas

* PETR4
* VALE3
* ITUB4
* BBDC4
* ABEV3
